# Time-Series-Forecasting-Champagne-Sales
Can historical monthly champagne sales be used to identify patterns and produce reliable forecasts for future sales?

This project analyses monthly champagne sales data spanning January 1964 to September 1972 to identify historical patterns and forecast future sales using time series techniques. The analysis covers exploratory data analysis, seasonal decomposition, and the development and evaluation of two forecasting models namely Exponential Smoothing (ETS) and SARIMA  compared against a held-out test period of 12 months.
